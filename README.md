
# Sextant Resilience Demo Console

## Overview

The Sextant Resilience Demo Console is a public-facing demonstration interface designed to showcase system visualization concepts for infrastructure resilience analysis.

This repository provides a lightweight frontend experience for exploring scenario-based system behavior through interactive dashboards and graphical representations.

---

## Purpose

This project is intended for:

- Demonstration of resilience visualization concepts
- Evaluation by technical stakeholders and partners
- Communication of system architecture at a high level
- UI/UX exploration of system state representations

---

## Features (Demo Layer Only)

- Interactive dashboard interface
- Scenario-based visualization views
- Graphical representation of system states
- Simulation-style UI behaviour (demo mode)
- Static or sample dataset rendering

---

## Important Note

This repository contains **only the presentation and visualization layer** of the Sextant platform.

The core system logic, including simulation engines, decision systems, and computational models, is maintained in a separate secured repository and is not included here.

---

## Architecture (High-Level)

The system follows a layered architecture approach:

- **Public Layer (this repository):** User interface and visualization components  
- **Secure Core Layer:** Protected computational and simulation systems  
- **Controlled Interface Layer:** API-based communication between systems (not exposed in this repository)

---

## Intended Use

This repository is intended for:

- Technical demonstration purposes
- Stakeholder review and evaluation
- Interface design reference
- Concept validation

It is not intended for production deployment or standalone execution of the full system.

---

## Access to Full System

Access to the full Sextant Resilience platform is restricted and may be provided under evaluation or partnership arrangements.

---

## Status

This is a **public demonstration prototype** and is part of an ongoing development and evaluation process.
